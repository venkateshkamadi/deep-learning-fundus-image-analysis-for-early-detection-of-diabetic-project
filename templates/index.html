About Project

Problem

Diabetic Retinopathy (DR) is a serious eye disease caused by diabetes. If not detected early, it can lead to permanent vision loss. Manual screening is time-consuming and depends on specialist availability.

Solution

This project uses Deep Learning and Transfer Learning techniques to automatically detect and classify Diabetic Retinopathy from retinal images, enabling faster and more reliable diagnosis.
